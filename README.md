# GTP Campus

Growth • Trust • Progress Campus

## Overview

GTP Campus is a Smart Campus Management Platform designed to simplify complaint management, application management, and communication between students, faculty, and administrators.

## Features

* Student Login
* Complaint Submission
* Anonymous Complaints
* Application Management
* Faculty Dashboard
* Admin Dashboard

## Technology Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python, Flask
* Database: MySQL
